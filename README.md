# Guia Digital Interativo: Cibersegurança para Idosos no Ambiente Mobile

Cartilha digital interativa desenvolvida para a **Atividade Extensionista II: Tecnologia Aplicada à Inclusão Digital** do curso de Análise e Desenvolvimento de Sistemas (Uninter).

O projeto ensina segurança digital básica para o público idoso, com foco em golpes aplicados por WhatsApp e redes sociais. Aplicado no **Lar São Vicente de Paulo**, em Taquarituba (SP).


## Tecnologias

- **HTML5** — estrutura e conteúdo da cartilha
- **CSS3** — design responsivo e identidade visual
- **JavaScript** — recursos de acessibilidade e quiz interativo

## Funcionalidades

- Aumentar e diminuir o tamanho da fonte (A+ / A-)
- Alternar entre tema claro e escuro
- Modo de alto contraste
- Quiz interativo para testar a identificação de riscos
- Layout responsivo (celular, tablet e computador)

## Como testar

Abra o arquivo `index.html` no navegador.

## Estrutura do projeto

| Arquivo | Descrição |
|---|---|
| `index.html` | Conteúdo da cartilha |
| `styles.css` | Identidade visual, temas e responsividade |
| `script.js` | Acessibilidade e quiz |
